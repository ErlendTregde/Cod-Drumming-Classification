# Cod Drumming Classification

## Setup

Install [uv](https://docs.astral.sh/uv/) if you don't have it already:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Then sync the environment (creates a venv and installs dependencies):

```bash
uv sync
```

## Structure

- `src/` — source code
- `notebooks/` — Jupyter notebooks
